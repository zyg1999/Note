 ### 下两栏固定, 中间自适应

#### 绝对定位

```css
.header,.footer{
    position:absolute;
    height:100px;
    width:100%;
}
.header{
	top:100px;
}
.footer{
    bottom:100px;
}
.main{
    position:absolute;
    width:100%;
    top:100px;
    bottom:100px;
    overflow:aoto;
}

```

#### flex 

```css
html,
body {
  padding: 0;
  margin: 0;
  height: 100%; /*定义页面整体高度为100%，重要*/
}
.container {
  display: flex;
  flex-direction: column; /*定义排列方向为竖排*/
  height: 100%; /*这里也要定义，重要*/
  width: 100%;
}
.header,
.footer {
  height: 100px;
  background-color: brown;
}
.main {
  height: auto;                                                                   
  flex: 1; /*中间分配剩下的所有空间*/
  background-color: burlywood;
  overflow: auto;
}
```
